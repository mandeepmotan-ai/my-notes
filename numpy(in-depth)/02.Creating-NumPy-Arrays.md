# Creating NumPy Arrays

### 1. Basic Array Creation (`np.array`)

- **1D Array:** Created from a simple Python list.
- **Multi-dimensional:**
    - **2D:** A list of lists (representing rows and columns).
    - **3D:** A list of nested lists (often used for image data or tensors).
- **dtype:** we can set a specific data type (e.g., `float`, `int`, `complex`) using the `dtype` parameter while creating a array using numpy.

```python
import numpy as np

# 1D, 2D, and 3D arrays
a = np.array([1, 2, 3])
b = np.array([[1, 2, 3], [4, 5, 6]])
c = np.array([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])

# Defining data type
float_arr = np.array([1, 2, 3], dtype=float)
```

### 2. Numerical Ranges & Reshaping

- **`np.arange(start, stop, step)`:** Similar to Python's `range()`, but returns an array.
- **`.reshape()`:** Changes the dimensions of an array without changing its data.
    - *Note:* The total number of elements must remain the same (e.g., `16` elements can be reshaped to `(2, 2, 2, 2)`).
- **`np.linspace(start, stop, num)`:** Generates `num` evenly spaced points between a start and stop value. Useful for plotting graphs.

```python
# Create range from 1 to 10 with step of 2
range_arr = np.arange(1, 11, 2) 

# Create and reshape to 4D
reshaped = np.arange(16).reshape(2, 2, 2, 2)

# Linear spacing
lin = np.linspace(-10, 10, 10, dtype=int)
```

### 3. Placeholder & Special Arrays

- **`np.ones()` / `np.zeros()`:** Creates an array filled entirely with 1s or 0s. Useful for initializing weights.
- **`np.random.random()`:** Creates an array of the given shape with random floats between `0.0` and `1.0`.
- **`np.identity()`:** Creates a square **Identity Matrix** (a matrix with 1s on the diagonal and 0s elsewhere).

```python
# Arrays with specific shapes (Rows, Columns)
ones = np.ones((3, 4))
zeros = np.zeros((3, 4))
rand = np.random.random((3, 4))

# 3x3 Identity Matrix
identity_mat = np.identity(3)
```
