# NumPy Array Attributes

To better understand how data is stored and structured in NumPy, we use **attributes**. These are properties of the `ndarray` object that describe its dimensions, size, and memory usage.

### 1. Dimension and Shape

- **`ndim` (Number of Dimensions):** Returns an integer representing the axes (dimensions) of the array.
    - *Example:* A 3D array returns `3`.
- **`shape`:** Returns a **tuple** representing the size of the array in each dimension (e.g., `(rows, columns)`).
    - For `a3`, a shape of `(2, 2, 2)` means it consists of 2 layers, each containing a 2x2 matrix.

```python
#example data
a1 = np.arange(10,dtype=np.int32)
a2 = np.arange(12,dtype=float).reshape(3,4)
a3 = np.arange(8).reshape(2,2,2)

# ndim: checks depth of array
a3.ndim # Result: 3

# shape: checks dimensions
print(a3.shape) # Result: (2, 2, 2)
```

### 2. Size and Memory

- **`size`:** The total number of elements in the array.
    - Calculated as the product of the elements of the `shape`.
- **`itemsize`:** The size (in **bytes**) of each individual element in the array.
    - *Note:* An `int32` or `float32` takes 4 bytes, while `int64` or `float64` takes 8 bytes.

```python
# size: total elements
print(a2.size) # Result: 12 (3 rows * 4 columns)

# itemsize: memory per element
a3.itemsize # Result: 8 (if default int64) or 4 (if int32)
```

### 3. Data Type

- **`dtype`:** Identifies the data type of the elements stored in the array.
- Since NumPy arrays are **homogeneous**, every element in the array will share this same type.

```python
print(a1.dtype) # Result: int32 (manually set)
print(a2.dtype) # Result: float64
print(a3.dtype) # Result: int64 (default for integers)
```
