# Essential NumPy Functions

This section covers the most frequently used NumPy utility functions for data manipulation, statistical analysis, and logic.

### 1. Data Modification (Sort, Append, Concatenate)

- **`np.sort()`**: Returns a sorted copy. Use `[::-1]` for descending order.
- **`np.append()`**: Adds values to the end of an array. If using a 2D array, specify the `axis`.
- **`np.concatenate()`**: Joins a sequence of arrays along an existing axis (similar to `vstack` or `hstack`).

```python
# Sorting b along columns
np.sort(b, axis=0) 

# Adding a column of random floats to b
np.append(b, np.random.random((b.shape[0], 1)), axis=1)

# Joining two arrays vertically
np.concatenate((c, d), axis=0)
```

### 2. Dimensionality & Shape

- **`np.unique()`**: Returns sorted unique elements of an array.
- **`np.expand_dims()`**: Expands the shape of an array by inserting a new axis at a specific position. Useful for matching dimensions for deep learning models.
- **`np.flip()`**: Reverses the order of elements along a given axis.

### 3. Logic & Search

- **`np.where(condition, [x, y])`**:
    - If only a condition is provided, it returns **indices**.
    - If $x$ and $y$ are provided, it acts as a **ternary operator**: Replace with $x$ if True, else $y$.
- **`np.argmax()` / `np.argmin()`**: Returns the **index** of the maximum or minimum value.
- **`np.isin(element, test_elements)`**: Returns a boolean array showing if each element is present in the provided list.

```python
# Logic: If > 50 set to 0, else keep original value
np.where(a > 50, 0, a)

# Index of the highest value
np.argmax(a)
```

### 4. Mathematical Aggregates

- **`np.cumsum()` / `np.cumprod()`**: Returns the cumulative sum or product.
- **`np.percentile()`**: Computes the $q$-th percentile (e.g., 50th percentile is the Median).
- **`np.histogram()`**: Computes the occurrences of data points within specific "bins."
- **`np.corrcoef()`**: Calculates the Pearson correlation coefficient between two variables (e.g., Salary vs. Experience).

### 5. Advanced Editing

- **`np.put()`**: Replaces specific elements at provided indices (works on a flattened version of the array).
- **`np.delete()`**: Returns a new array with sub-arrays at specific indices removed.
- **`np.clip()`**: Limits the values in an array. Any value below `min` becomes `min`, and any above `max` becomes `max`.

```python
# Keep all values between 25 and 75
np.clip(a, a_min=25, a_max=75)
```

---

### 6. Set Operations

Used for finding relationships between two 1D arrays:

- **`np.union1d(m, n)`**: Unique elements in both.
- **`np.intersect1d(m, n)`**: Common elements.
- **`np.setdiff1d(m, n)`**: Elements in $m$ but not in $n$.
- **`np.setxor1d(m, n)`**: Elements in either $m$ or $n$, but not both.

### Summary Table for Notion

| **Function** | **Primary Use Case** |
| --- | --- |
| **`np.where`** | Conditional replacement or finding indices. |
| **`np.argmax`** | Finding the location of the highest value. |
| **`np.unique`** | Finding distinct values in a dataset. |
| **`np.cumsum`** | Running totals (e.g., daily revenue). |
| **`np.clip`** | Removing outliers by capping values. |
| **`np.corrcoef`** | Checking how two variables relate. |
