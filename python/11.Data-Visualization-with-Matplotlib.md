# Data Visualization with Matplotlib

# Matplotlib

- **Matplotlib** is the fundamental plotting library in Python.
- Works with **NumPy arrays + Pandas DataFrames**.
- Very customizable (low-level control of every element).
- Most other libraries (Seaborn, Pandas plots) are built on top of it.

ðŸ‘‰ Think of it as the "Photoshop of Python plots" â€” powerful but needs practice.

### import and setup

```kotlin
import matplotlib.pyplot as plt
import numpy as np
```

### Basic Plot

```kotlin
x = np.linspace(0, 10, 100)   # 100 points between 0-10
y = np.sin(x)

plt.plot(x, y)
plt.show()
```

### Labels, Titles and Legends

```kotlin
plt.plot(x, y, label="sin(x)", color="blue", linestyle="--")
plt.title("Sine Function")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.legend()
plt.grid(True)
plt.show()
```

### Multiple Plots

```kotlin
y2 = np.cos(x)
plt.plot(x, y, label="sin(x)")
plt.plot(x, y2, label="cos(x)")
plt.legend()
plt.show()
```

### Scatter Plots

```kotlin
np.random.seed(0)
x = np.random.rand(50)
y = np.random.rand(50)
plt.scatter(x, y, c='red', marker='o')
plt.title("Random Scatter")
plt.show()
```

### Bar Charts

```kotlin
categories = ["Math", "Science", "English"]
scores = [85, 90, 78]

plt.bar(categories, scores, color='skyblue')
plt.title("Student Scores")
plt.show()
```

### Histogram

```kotlin
data = np.random.randn(1000)  # normal distribution
plt.hist(data, bins=30, color="green", alpha=0.7)
plt.title("Histogram of Normal Distribution")
plt.show()
```

### Pie Chart

```kotlin
sizes = [30, 25, 20, 25]
labels = ["A", "B", "C", "D"]

plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=140)
plt.title("Pie Chart Example")
plt.show()
```

### Styling and Customization

```kotlin
plt.style.use("ggplot")   # other: 'seaborn', 'classic', 'dark_background'

x = np.linspace(0, 10, 100)
y = np.exp(x)

plt.plot(x, y, linewidth=2, color="purple")
plt.title("Exponential Growth", fontsize=16)
plt.show()
```

### With Pandas

```kotlin
import pandas as pd

data = {
    "Name": ["Alice", "Bob", "Charlie", "David"],
    "Math": [85, 90, 78, 92],
    "Science": [91, 85, 89, 95]
}
df = pd.DataFrame(data)

df.plot(x="Name", y=["Math","Science"], kind="bar", figsize=(6,4))
plt.title("Student Scores")
plt.show()
```
